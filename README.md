# 色名クイズ (Color Name Quiz)

色の解説文を読んで、その「色名」と「系統色名（JIS慣用色名など）」を当てるWebブラウザ用クイズアプリです。


## 🎨 特徴

* **ランダム出題**: 登録されたデータからランダムに出題されます（一度出題された問題は重複しません）。
* **詳細な判定**: 「色名」と「系統色名」の両方の入力が必要です。それぞれ個別に正誤判定が行われます。
* **結果表示**: 回答後に、正解の色見本（RGB）、マンセル値、解説が表示されます。
* **成績管理**: いつでも「成績・結果を見る」ボタンから、これまでの回答履歴と正解率を確認できます。
* **レスポンシブ対応**: PCおよびスマートフォンでの動作を想定しています。

## 💻 ローカルでの実行方法

このアプリは外部JSONファイル（`colordata.json`）を読み込む仕組みのため、ローカルPC上のファイルを直接ブラウザで開く（`file://` プロトコル）と、セキュリティ制限（CORSエラー）により動作しません。

以下のいずれかの方法で実行してください。

### 推奨: VS Code (Visual Studio Code) を使用する場合

1.  このリポジトリをクローン、またはダウンロードしてVS Codeで開きます。
2.  拡張機能「**Live Server**」をインストールします。
3.  `index.html` を右クリックし、**"Open with Live Server"** を選択します。
4.  ブラウザが起動し、アプリが動作します。

### その他のローカルサーバーを使用する場合

Pythonがインストールされている環境であれば、フォルダ内で以下のコマンドを実行して簡易サーバーを立ち上げることができます。

```bash
# Python 3.x
python -m http.server 8000
